Projects

Student Registration Form

I developed a Student Registration Form using HTML, designed to collect essential student details in an organized manner. The form includes fields for First Name, Last Name, Date of Birth, Email ID, Mobile Number, Address, City, State, Country, and PIN Code. It also features radio buttons for gender selection, checkboxes for hobbies, and a table for educational qualifications where students can input their board, percentage, and year of passing. Additionally, students can select their preferred course (BCA, B.Com, B.Sc, or B.A.) using radio buttons. The form is structured with proper labels and input constraints, ensuring a clean and user-friendly interface. A Submit button is included for form submission, along with a Reset button to clear all fields. This project helped me strengthen my understanding of HTML form elements, tables, and input validation, making it a fundamental step in web development.

Gambia Flag

I created the flag of Gambia using HTML and CSS, demonstrating my skills in structuring and styling web elements. The flag consists of three horizontal stripes: red at the top, blue in the center with white borders, and green at the bottom. I used HTML div elements to structure the flag and applied CSS for precise colors, width, height, and alignment. By utilizing flexbox and background properties, I ensured a clean and responsive design. This project enhanced my understanding of CSS positioning, color theory, and layout techniques, reinforcing my ability to create visually accurate designs using web technologies.
